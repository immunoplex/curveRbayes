Columns in ensemble[[model]]$grid.
Column Description
curve_id Integer FK — identifies which calibration curve this row belongs to.
concentration log10(AU/mL) grid point — the x-axis value on the preprocessed scale.
pred_mean Posterior mean of the forward-predicted response at this concentration.
pred_sd Posterior SD of the forward-predicted response (parameter uncertainty only).
pred_noise_sd Total predictive SD after Student-t noise injection (parameter uncertainty + measurement noise).
cv_pct % CV of back-calculated concentration at this grid point.
model Model identifier.
